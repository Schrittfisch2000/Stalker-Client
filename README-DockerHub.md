# Docker Hub Kurzreferenz

Das UGREEN-System nutzt das veröffentlichte Docker-Image:

```yaml
image: schrittfisch2000/stalker-client:latest
pull_policy: always
```

Neue Versionen erscheinen nach einem Release-Tag automatisch auf Docker Hub.

Beispiel:

```text
v1.0.30
→ schrittfisch2000/stalker-client:v1.0.30
→ schrittfisch2000/stalker-client:latest
```
